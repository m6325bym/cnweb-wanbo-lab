# cnweb-wanbo-lab

本仓库用于归档和发布多个独立的 HTML 页面，主要用于存储和展示各类网页资源。

## 项目简介

这是一个轻量级的静态页面归档仓库，存放了若干独立开发的 HTML 文件。每个页面均为自包含的完整网页，可直接在浏览器中打开运行。仓库不针对任何特定域名或网站，仅作为个人或协作项目的页面集合。

## 目录说明

```
/
├── index.html          # 仓库主页（可选）
├── pages/              # 归档页面目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下存放各个独立的 HTML 页面，文件名可反映页面内容或用途。
- 根目录的 `index.html` 为可选的导航页面，便于浏览各归档页面。

## 页面归档说明

- 每个 HTML 页面均为完整、独立的网页，不依赖外部资源（除非页面自身引用）。
- 页面内容可能包括实验性设计、工具页面、展示页面等，具体用途见页面内部注释。
- 所有页面均可通过直接打开 HTML 文件或在 GitHub Pages 等静态托管服务上访问。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或更新现有页面。
- 页面内容可能随时调整，不保证向后兼容。
- 欢迎提交 Issue 或 Pull Request 进行讨论或贡献，但请注意仓库定位为个人归档用途。
